⇦Puzzle Game

How to play the puzzle game?

  1. Select an image using next/previous buttons.
  2. After you decided on a picture, now you can hide it using the hide/show image button.
  3. Now you can solve the puzzle, reconstructing the original selected image, by swapping image pieces position.

    Click on one piece, this will be highlighted by a red border and then click on the position where you want to move the highlighted piece, these two pieces will be swapped. The swaps will be counted in the header near the show/hide image button.

  4. Enjoy the game!

    The images used in this puzzle game are hosted and curated by Pexels. Please visit Pexels website for more beautiful and inspiring photos
